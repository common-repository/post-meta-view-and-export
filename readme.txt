This plugin adds possibility to view and export post metadata for superadmins.

== Description ==

Plugin adds section to post edit page in WordPress admin, where the post metadata are displayed in convenient table with the option to export the data into JSON format

== Installation ==

1. Install the plugin from wordpress.org repository
